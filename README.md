# dotnet


To create sln file — 

```
dotnet new sln [options]

dotnet new sln -name myProject
```


To add project to sln we use following command —-

```

dotnet sln [slnFiIeName] add [csproj-file-path ]

dotnet sln myProject.sln add Web/Web.csproj
```
